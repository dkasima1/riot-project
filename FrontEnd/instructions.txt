Installing Project Locally
1) Install Node 12.18.2 and Yarn
2) git clone https://github.com/dkasima1/riot-project.git
3) npm install
4) yarn add react-router-dom
5) yarn add react-bootstrap bootstrap
6) yarn add styled-components
7) npm install --save mdbreact
8) npm install -g json-server 
9) json-server --watch public/db.json --port 3001
(If no permission run in windows powershell: Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Unrestricted)
10) npm i axios --save


Git Branch Control
1) git fetch
2) git checkout -b <Name of local Branch tracking remote> origin/<remote branch name>
